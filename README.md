# 📊 Stock Market Analysis Dashboard (Business Analyst Project)

## 🔍 Project Overview

This project presents a comprehensive analysis of stock market performance using historical data of leading technology companies, including Apple, Microsoft, Netflix, and Google. The objective is to derive actionable insights through data visualization and analytical techniques using Tableau.

The dashboard enables users to explore trends, compare company performance, and understand relationships between stock prices to support data-driven decision-making.

---

## 🎯 Objectives

* Analyze stock price trends over time
* Compare performance across multiple companies
* Calculate and visualize moving averages
* Measure volatility in stock prices
* Perform correlation analysis between stocks
* Identify patterns and anomalies in market behavior

---

## 📁 Dataset Description

The dataset consists of historical stock price data for the following companies:

* Apple
* Microsoft
* Netflix
* Google

### Key Attributes:

* Date
* Open Price
* Close Price
* High
* Low
* Volume

---

## 🧹 Data Preparation & Cleaning

* Removed inconsistencies and missing values
* Standardized date formats
* Ensured correct data types for numerical fields
* Structured dataset for time-series analysis
* Performed preprocessing using Excel and SQL

---

## 🛠 Tools & Technologies

* Tableau Desktop
* Microsoft Excel

---

## 📈 Key Insights

* Stock prices exhibit clear trends influenced by market conditions
* Technology companies show varying levels of volatility
* Strong correlations observed between certain stocks
* Moving averages help in identifying stable long-term patterns
